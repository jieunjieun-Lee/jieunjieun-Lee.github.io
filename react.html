React.js 작업물 소개

React.js를 배우면서 선생님의 도움을 받아 처음으로 작업한 결과물입니다.
대기자 명단 입력창에 이름을 입력하고 '확인' 버튼을 누르면 명단 맨 아래에 추가됩니다.
명단에 있는 이름을 클릭하면 빨간 취소선이 그어지게끔 표현했고, '승인' 버튼을 누르면 해당 이름이 삭제됩니다.
각각의 컴포넌트를 Main.js, Input.js, List.js, Item.js 4가지로 나누어서 작업했습니다.
생각보다 더 세세하게 컴포넌트가 나뉘어진다는 것을 알게 된 작업이었습니다.

  • React 작업 사이트

React.js를 사용해 처음으로 만들어 본 사이트입니다.
header, visual, content, footer별로 폴더를 만들고 그 안에서 작업이 이루어졌습니다.
다른 폴더들은 컴포넌트 하나로 작업했지만 content폴더의 경우, Contents.js, Menu.js, ConBox1.js 3개의 컴포넌트로 나누어 작업했습니다. MainVisual.js 부분에는 플러그인을 사용해 표현했습니다.
컴포넌트별로 분리해서 이루어지는 작업이 html과 가장 큰 차이점으로 다가왔습니다. 또한, css 역시 폴더별로 작업이 이루어진다는 점이 신선하게 다가왔습니다.
혼자가 아니라 선생님의 도움을 받아 만든 작업물이긴 하지만, React.js로 사이트를 만드는 작업이 어떤 식으로 이루어 지는지 알게 된 소중한 경험이 되었습니다.
